# 贪吃蛇游戏 (Snake Game)

这是一个基于 HTML5 Canvas、CSS3 和 JavaScript 开发的经典贪吃蛇网页游戏。

## 🎮 游戏特色

*   **经典玩法**：控制贪吃蛇吃掉红色食物，每吃一个身体变长，速度根据设置调整。
*   **自定义速度**：提供 1-20 级速度调节滑块，支持游戏过程中实时变速。
*   **响应式设计**：完美适配桌面端（键盘控制）和移动端（触控按钮）。
*   **记分系统**：实时记录当前得分，并自动保存最高分记录到本地浏览器。
*   **音效反馈**：使用 Web Audio API 生成复古的吃豆和死亡音效。

## 🕹️ 操作说明

### 电脑端
*   **开始/重置**：点击“开始游戏”或“重新开始”按钮。
*   **移动**：使用键盘方向键 `↑` `↓` `←` `→` 控制移动方向。
*   **速度调节**：拖动滑块调整游戏速度（1=最慢，20=最快）。

### 移动端
*   **移动**：点击屏幕下方的虚拟方向按钮控制移动。
*   **其他**：与电脑端一致。

## 🚀 如何运行

本游戏是纯静态网页，无需复杂的后端环境。

### 方法 1：直接打开
双击项目目录下的 `index.html` 文件即可在浏览器中运行。

### 方法 2：本地服务器 (推荐)
为了获得最佳体验（避免某些浏览器的安全策略限制），建议使用本地服务器运行。

如果你安装了 Python：
```bash
python -m http.server 8080
```
然后在浏览器访问：[http://localhost:8080](http://localhost:8080)

如果你安装了 Node.js：
```bash
npx http-server -p 8080
```
